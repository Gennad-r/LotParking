Стойка парковочная въездная

Парковочная въездная стойка специально предназначена для обслуживания контролируемого въезда автомобилей на платную парковку. Въездная стойка обеспечивает выдачу бесконтактных карт, предназначенных для последующей оплаты согласно тарифам. Обеспечивает работу с VIP картами. Устанавливается на въезде на территорию парковки перед шлагбаумом.

Стойка парковочная выездная

Парковочная выездная стойка специально разработана для обслуживания контролируемого выезда автомобилей с территории платной парковки. Выездная стойка предназначена для приема бесконтактных карт и контроля оплаты парковки. Обеспечивает работу с VIP картами. Устанавливается на выезде с территории парковки перед шлагбаумом.
